# Ejecutar en desarrollo

1. clonar el repositorio
2. instalar dependencias con 
```
pnpm install
```
3. Instalar NestJS CLI
```
npm i -g @nestjs/cli
```
4. Levantar la base de datos
```
docker-compose up -d
```

5. reconstruir la base de datos
```
http://localhost:3000/api/v2/seed
```
6.
